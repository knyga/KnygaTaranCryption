После выбора шифра и режима вы увидите следующее окно:


  1. Нажмите на кнопку "Редактировать текстово"
  2. Наберите нужный текст
  3. Нажмите на кнопку "Сохранить"

Получим:


Зададим параметры ключа. Для этого выбираем в меню "Правка"->"Опции шифрования"

  1. Пишем фразу-пароль, на основе которой будет сгенирирован ключ
  2. Выбираем "шаг" для псевдно-случайной генерации вектора инициализации IV. Это должно быть целое число
  3. Выбираем длину ключа, если соответствующий выбор есть
  4. Для просмотра вектора IV в бинарном виде нажимаем "Просмотр"
  5. Для сохранения параметров нажимаем "Сохранить"

Рекомендуется включить опцию "Авто-кратность" во вкладке "Правка"


Для шифрования открытого текста нажимаем кнопку "Зашифровать"

Для смены кодировок и просмотра сообщений в бинарном виде нажимаем соответственно "Кодировка" и "Редактировать бинарно". Для копирования открытого текста в область зашифрованного нажимаем "В расшифровку", и наоборот - "В зашифровку"
